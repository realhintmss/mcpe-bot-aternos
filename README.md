# Minecraft Bedrock Bot For Keep Server On

## Version compatibility

`bedrock-protocol` may lag behind the newest Bedrock patch versions (for example `1.26.23`).

Set `server_version` in `config.json` to `"auto"` (default) to let the bot pick a compatible protocol version automatically.

If you set a specific unsupported version, the bot now retries once with automatic version detection.
